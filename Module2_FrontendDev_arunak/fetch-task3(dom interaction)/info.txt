🧠 Core Concepts You’re Learning
1. HTML Forms

You collect user data:

<form>
  <input type="text" id="name" placeholder="Enter your name">
  <button type="button" onclick="showMessage()">Submit</button>
</form>
2. DOM (Document Object Model)

This is how JavaScript talks to HTML

document.getElementById("name")

👉 Means: “Go find that element on the page”

3. Event Handling

You respond to actions like:

clicks
typing
submitting
onclick="showMessage()"

👉 Means: “When clicked → run this function”

4. Changing the Page Dynamically
element.innerText = "Hello!"

👉 You are literally editing the webpage live
